# ✅ To-Do List Application

## Overview

The **To-Do List Application** is a dynamic, user-centric task management tool built with PHP, JavaScript, and MySQL. It empowers users to efficiently manage their tasks by providing functionalities such as adding, editing, deleting, and marking tasks as completed. The application features a responsive design, ensuring seamless usability across devices.

## 🔧 Features

- **Task Management**: Add, edit, delete, and mark tasks as completed.
- **Persistent Storage**: Utilizes MySQL database for storing tasks.
- **Responsive Interface**: Ensures optimal user experience on both desktop and mobile devices.
- **User Authentication**: Secure login and registration system for personalized task management.

## 🚀 Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Version Control**: Git & GitHub

## 📦 Installation

To set up this application locally:

1. Make sure you have **XAMPP** (or any PHP server like WAMP, MAMP) installed and running.
2. Clone the repository:
   ```bash
   git clone https://github.com/nanishat/todolist.git
   ```
3. Copy the project folder into your htdocs directory (for XAMPP) or the equivalent web root folder for your server.
  ```bash
   C:\xampp\htdocs\todolist
   ```
4. Import the database schema from database.sql into your MySQL database.

5. Configure the database connection in config.php with your database credentials.

6. Open your browser and navigate to:
  ```bash
   http://localhost/todolist/index.php
   ```

## 🛠️ Usage

Open the application in your browser.

Register for a new account or log in with your existing credentials.

Begin managing your tasks by adding, editing, or deleting them as needed.

Mark tasks as completed to keep track of your progress.
